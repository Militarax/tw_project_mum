1. Abstract

Music Review Manager este o aplicatie web care permite specificarea adnotarilor textuale si a meta-datelor , exprimarea opiniilor vizand auditiile muzicale realizate de o persoana sau grup de persoane.

2. Descriere

Aplicatia va realiza managementul productiilor muzicale, inclusiv organizarea lor pe diverse criterii. Dupa autentificare utilizatorul isi poate exprima opinia vizand auditiile muzicale realizate de o persoana sau grup de persoane. Se vor ofera statistici diverse ce pot fi exportate in formate deschise – minimal, CSV si PDF. Diversele anunturi (o persoana a comentat un album muzical, a aparut o alta compilatie de interes, modificarea clasamentului celor mai apreciate cantece ale lunii etc.) vor fi expuse ca flux de stiri RSS. Listele de melodii preferate vor fi importate/exportate in format JSPF/XSPF.

3. Componente

3.1 Login / Register

Utilizatorii se vor putea autentifica sau inregistra. Logarea se face pe baza de email/parola.

3.2 Statistica

Utilizatorii pot descarca statistici in format csv si pdf.

3.3 Artist

Vizualizare date artist si melodiile acestuia.

3.4 Album

Lista melodii album.

3.5 RSS

Vizualizare evenimente ce au avut loc.

3.6 Comentarii

Adaugarea comentariilor pentru un album,track.

3.7 Exportare/Importare

Exportare/Importare unui playlist in format JSPF.

5. Detalii de implementare

5.1 Baza de date

  1. Tabela users ( id, email, password(md5) ) contine toti userii registrati.
  2. Tabela tracks ( id, id_artist, title_track ) contine toate melodiile.
  3. Tabela artists ( id, name, wiki, real_name, born, genre, img_link ) contine toti artistii care au macar o melodie.
  4. Tabela albums ( id, album_title, id_track, id_artist, img_link ) contine toate albumelea unui artist.
  5. Tabela rss ( id, id_title, link, description, date ) retine toate evenimentele care au avut loc pe site.
  6. Tabela track_list ( id_user, id_track ) retine playlist-ul persoanei care este autentificata.
  7. Tabela (play/artist/album/personal)comments ( id, id_user, album_title, comment, date) contine toate commentariile despre o melodie/artist/album/user.
